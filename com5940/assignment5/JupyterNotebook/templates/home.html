Airtable Data Source

The data used in the dashboard demonstration can be found in this Airtable database. The chart receives its data from the "Roll-up" table and the map from the "Venues" table. The album data is stored in MySQL datase. You can try add, update, or delete products from the album by using the add, update, and delete buttons above the display grid.

The dataset was originally derived from a tutorial by d3-media on learning Refine for data cleaning and transformation. The original link is here.
